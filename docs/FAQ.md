# FAQ

### Почему выбрана модель Qwen и vLLM?
Локальный запуск через vLLM обеспечивает совместимость с OpenAI API и
оптимальное соотношение скорость/качество для русскоязычных задач.

### Как добавить новый навык или индикатор в JD?
Добавьте его в JSON JD (`competencies[*].indicators`) и, при необходимости,
обновите FAISS‑индекс. Вес компетенции должен быть скорректирован так, чтобы
сумма оставалась 1.0.

### Как переключить HR_EMB_BACKEND?
Установите переменную окружения `HR_EMB_BACKEND` (`BGE_M3`, `CONSULTANTBERT` или
`TAROT`) и перезапустите сервис.

### Где посмотреть таймкоды evidence?
В ответе `/rubric/score` поле `evidence` содержит `t0`/`t1`; в отчёте они
доступны как ссылки на `audio?t0=...&t1=...`.

### Что влияет на латентность?
- размер аудио‑чанков и VAD
- скорость LLM (модель, max_tokens)
- наличие прогрева моделей (vLLM, TTS)

### Как добавить новый «skill» для DM?
Обновите JD и промпт диалогового менеджера; функция `retrieve_indicators` уже
использует JD для RAG‑подсказок.
